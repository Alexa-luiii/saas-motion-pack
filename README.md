# SaaS Motion Pack – 15 Animated Tailwind Components

**A premium dark-theme Tailwind CSS component pack built for AI startups and SaaS projects. Fully responsive, modern, and interactive UI sections ready to use.**

---

## 🔹 What’s Included

This pack contains **15 ready-to-use HTML components**, all in **dark theme** and designed for enterprise SaaS interfaces:

1. **Hero Sections**
   - `hero-1.html` – Enterprise AI Infrastructure
   - `hero-2.html` – Automate Intelligence at Scale
   - `hero-3.html` – Transform Data into Strategic Advantage

2. **Navbar Sections**
   - `navbar-1.html` – Corporate AI Navigation
   - `navbar-2.html` – Responsive AI Platform Menu

3. **Pricing Sections**
   - `pricing-1.html` – Corporate AI Pricing
   - `pricing-2.html` – Corporate AI Pricing Variant

4. **Features / CTA / Modals**
   - `features-1.html` – Core AI Capabilities
   - `features-2.html` – Built for Enterprise Performance
   - `features-3.html` – Advanced Automation Solutions
   - `cta-1.html` – Corporate AI Call-to-Action
   - `cta-2.html` – Accelerate Your AI Projects
   - `modal.html` – Reusable modal component with open/close functionality
   - `accordion.html` – Interactive FAQ / features accordion
   - `button-loader.html` – Animated button with loader state

---

## 🔹 Features

- **Dark Theme Design** – Modern dark UI for AI startups and SaaS products  
- **Tailwind CSS v3.x** – Easy to integrate, no heavy frameworks required  
- **Fully Responsive** – Works on mobile, tablet, and desktop  
- **Interactive Components** – Buttons, modals, accordion, loaders, and CTAs with smooth animations  
- **Ready to Use** – Just copy the HTML into your project or edit as needed  
- **Customizable** – Fonts, colors, gradients, and content can be easily adjusted  

---

## 🔹 How to Use

1. **Include Tailwind CDN** (already included in each component) or install via npm in your project.  
2. **Open the component HTML file** in your editor.  
3. **Copy the section** into your project or run standalone to preview.  
4. **Customize** headings, buttons, and images as needed.  
5. **Preview**: Open in a browser and see responsive animations.
